# Digital film shelf
Digital Film Shelf is a web application for creating and managing a personal film collection. Users can register, log in, add films to their collection, and browse public film collections from other users.

The project is built with HTML, CSS, JavaScript and Supabase.

## Features
- User registration and login
- Add films to a personal collection
- View your own film collection
- Browse public collections from other users
- Sort films by title, year, genre and format
- Store film details such as title, director, genre, format, release year and cover image

## Planned Features
- Edit and delete films
- Search and filter films
- User profile pages
- Personal ratings and notes
- Public/private collection settings
- Collection statistics

## Tech Stack
